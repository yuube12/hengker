📌 Apa Itu OpenVPN?

  • openvpn adalah aplikasi client untuk menjalankan koneksi VPN.
  • Biasanya sudah terinstal di Kali Linux. Cek dengan:
    openvpn --version
  • Kalau hanya mengetik openvpn, akan muncul manual-nya.
Tapi OpenVPN tidak bisa digunakan langsung. Kamu perlu:
  • Konfigurasi file .ovpn dari penyedia VPN (seperti VPNBook)
  • Username dan password (jika diperlukan)

🔧 Langkah-Langkah Menggunakan VPNBook di Kali Linux

  1. Kunjungi situs: vpnbook.com
  1. Klik tab "Free VPN" di atas
  1. Scroll ke bagian "OpenVPN Config Files"
  1. Download beberapa file ZIP, misalnya:
    • vpnbook-us1-tcp80.ovpn.zip
    • vpnbook-ca1-udp25000.ovpn.zip, dll

    Karena kadang satu server penuh

  1. Ekstrak file ZIP-nya
  1. Buka terminal dan masuk ke folder hasil ekstrak:
    cd ~/Downloads/vpnbook-.../
  1. Jalankan file .ovpn:
    sudo openvpn vpnbook-(tulis namanya).ovpn
  1. Masukkan username: vpnbook
  1. Password bisa kamu lihat di halaman yang sama (bagian bawah). Selalu berubah, jadi harus cek ulang.
  1. Jika berhasil, akan muncul:
    Initialization Sequence Completed

    Artinya: VPN aktif

  1. Kalau tidak bisa, berarti server penuh. Hentikan openvpn dengan: (di terminal lain)
    sudo pkill openvpn

    Perintah ini akan mencoba menghentikan semua proses bernama openvpn.

  1. Coba hubungkan lagi hingga berhasil.
  1. Setelah berhasil, coba akses internet, misalnya buka Google. Jika tidak bisa, lakukan perbaikan DNS.

❌ Cara Berhenti dari VPN

  • Tekan Ctrl + C di terminal tempat VPN berjalan.
  • langsung sudo pkill openvpn kalau tidak bisa ditekan.

🔁 Cara Mengganti DNS di Kali Linux

File DNS default adalah /etc/resolv.conf, namun biasanya dilindungi dengan atribut "immutable" agar tidak bisa diubah seenaknya.
  1. Cek status proteksi file:
    sudo lsattr /etc/resolv.conf
    Jika ada huruf i, artinya file dikunci.
  1. Hapus proyeksi
    sudo chattr -i /etc/resolv.conf
  1. Edit file DNS
    sudo nano /etc/resolv.conf
  1. Tambahkan dengan DNS yang kamu pilih
    nameserver 8.8.8.8       # Google DNS
nameserver 8.8.4.4       # Google DNS cadangan
nameserver 1.1.1.1       # Cloudflare DNS (lebih aman dan privat)

    Lalu berikan DNS asli tanda pagar #

  1. Simpan dengan:
  • Ctrl + O, lalu Enter
  • Ctrl + X untuk keluar

❗ PERINGATAN:

  • DNS akan kembali ke asli saat reboot (kecuali kamu ubah permanen).

⚠️ Harus pakai VPN, kenapa?

  • Beberapa ISP memblokir akses ke situs Tor atau mendeteksi lalu lintas Tor.
  • VPN menyamarkan identitasmu sebelum kamu membuka koneksi Tor.
  • Tidak wajib 100%, tapi sangat disarankan demi privasi.

📥 Langkah-Langkah Instalasi Tor Browser:

  1. Aktifkan VPN terlebih dahulu.
  1. Buka Firefox (atau browser lain), dan akses situs resmi:

    https://www.torproject.org

  1. Klik tombol Download Tor Browser for Linux.
  1. File yang terunduh biasanya berekstensi .tar.xz.
  1. Ekstrak file tersebut (klik kanan → Extract Here, atau gunakan terminal: tar -xf <namafile>).
  1. Masuk ke folder hasil ekstraksi, biasanya bernama tor-browser_en-US/.

▶️ Menjalankan Tor Browser:

Cara 1: GUI (jika berhasil)

7. Double klik file shortcut tor di folder tersebut.

  • Tapi kemungkinan besar ini tidak akan berhasil.

Cara 2: Terminal (lebih stabil)

8. Buka terminal, masuk ke folder, lalu jalankan:

./start-tor-browser.desktop

🌐 Mengatur Koneksi (Pertama Kali)

  1. Akan muncul dua pilihan: Connect dan Configure.
  1. Klik Configure jika:
    • Koneksi lama terlalu lambat
    • Kamu berada di jaringan yang memblokir Tor
  1. Pada pengaturan, aktifkan:
    • Select a built-in bridge
    • Pilih jenis: obfs4

    (Bridge ini menyamarkan koneksi Tor agar tidak terdeteksi ISP.)

  1. Klik Connect.
  1. Tunggu beberapa saat. Jika berhasil, Tor Browser akan terbuka.
