Apa Itu Networking

Dalam dunia jaringan komputer, semua perangkat yang terhubung, seperti komputer, laptop, printer, bahkan HP, biasanya tergabung dalam sebuah jaringan. Secara umum, jaringan itu dibagi menjadi dua jenis: private network dan public network.

üè† Private Network

Private network itu bisa kita anggap seperti jaringan internal atau pribadi. Contohnya:

  • Jaringan Wi-Fi di rumah
  • Jaringan komputer di sekolah atau kampus
  • Jaringan internal di kantor

Dalam jaringan ini, hanya perangkat-perangkat tertentu saja yang bisa bergabung. Biasanya digunakan untuk berkomunikasi dan berbagi data secara lokal, seperti mencetak dokumen ke printer atau berbagi file antar komputer.

Private network tidak langsung terbuka ke dunia luar (internet). Jadi, secara keamanan juga lebih terjaga.

üåç Public Network

Public network adalah kebalikannya. Ini adalah jaringan yang terbuka untuk umum dan digunakan untuk menghubungkan banyak private network di seluruh dunia. Public network inilah yang menjadi dasar dari internet.

Contohnya:

  • Saat membuka Google atau YouTube dari rumah
  • Saat perusahaan terhubung ke server di negara lain
  • Ketika menggunakan hotspot Wi-Fi di tempat umum

Semua itu terjadi lewat public network.

üîó Hubungan Keduanya

Bayangkan private network itu seperti rumah-rumah pribadi. Nah, jalanan besar yang menghubungkan rumah-rumah itu adalah public network. Kalau kamu ingin mengirim surat dari rumahmu ke rumah temanmu, surat itu harus lewat jalanan umum dulu.

Begitu juga dengan data. Saat kamu mengakses situs web, data dari komputer kamu (di private network) akan melewati public network (internet), lalu sampai ke server web yang mungkin juga berada di private network lain.

üåê 1. IP Address

IP Address = alamat numerik (angka) yang digunakan untuk mengidentifikasi perangkat di jaringan.

Contoh IP: 192.168.1.1

üì¶ IP dibagi menjadi 4 bagian (oktet). Misalnya:

Oktet 1: 192
Oktet 2: 168
Oktet 3: 1
Oktet 4: 1

Jumlah kombinasi IP di sistem ini disebut IPv4, yang hanya bisa menampung sekitar 4,29 miliar alamat.

üîí Private IP Address

  • Digunakan di dalam jaringan lokal (rumah, sekolah, kantor)
  • Tidak bisa langsung digunakan untuk akses internet
  • Biasanya seperti:
    • 192.168.x.x
    • 10.x.x.x
    • 172.16.x.x ‚Äì 172.31.x.x

Contoh:

Mereka bisa saling terhubung di jaringan lokal.

üåç Public IP Address

  • Digunakan untuk menghubungkan ke Internet
  • Diberikan oleh ISP (Internet Service Provider)
  • Satu rumah biasanya berbagi satu public IP
  • Bisa terlihat sama pada banyak perangkat di rumah

üßæ 2. MAC Address

MAC Address (Media Access Control Address) adalah identitas unik yang diberikan ke setiap perangkat jaringan, seperti komputer, laptop, atau smartphone. MAC Address diberikan saat perangkat diproduksi di pabrik, dan digunakan untuk mengidentifikasi perangkat secara fisik di jaringan lokal (LAN).

üß© Bentuk MAC Address:

MAC Address terdiri dari 12 karakter heksadesimal (angka dan huruf A‚ÄìF) dan ditulis dalam 6 pasang, dipisahkan dengan tanda titik dua :.

Contoh:

a4:c3:f0:85:ac:2d

üì¶ Struktur:

Bagian MAC AddressFungsi
a4:c3:f0Kode perusahaan pembuat perangkat (misalnya: Intel)
85:ac:2dNomor unik untuk perangkat tersebut

üß† Fungsi MAC Address

MAC Address digunakan oleh perangkat jaringan lokal (router, switch, komputer) untuk mengirimkan data ke perangkat yang benar dalam satu jaringan. Di jaringan lokal, komunikasi sering terjadi antar MAC Address, bukan langsung antar IP Address.

üé≠ MAC Address Spoofing (Penyamaran MAC)

‚ùìApa Itu?

Spoofing adalah proses di mana seseorang mengubah MAC Address perangkatnya agar terlihat seperti perangkat lain.

üß™ Contoh Kasus:

Misalnya:

  • Sebuah firewall hanya mengizinkan komunikasi dari MAC Address milik administrator.
  • Seorang penyerang mengganti MAC Address-nya agar terlihat sama seperti administrator.
  • Firewall sekarang mengira penyerang adalah admin dan mengizinkan akses penuh!

‚û°Ô∏è Hasilnya: sistem keamanan rusak.

‚òï Contoh di Dunia Nyata: Wi-Fi di Kafe & Hotel

Beberapa tempat umum seperti kafe, hotel, atau ruang publik menggunakan sistem yang:

  • Mengenali perangkat berdasarkan MAC Address
  • Misalnya, satu perangkat = gratis, lebih dari itu = harus bayar

Penyerang bisa:

  • Mengganti MAC Address
  • Menyamar sebagai perangkat lain
  • Mendapat akses gratis atau premium tanpa izin

Ping adalah alat dasar dalam jaringan komputer untuk mengetes koneksi antara dua perangkat. Tujuannya untuk mengetahui:

  • Apakah perangkat lain bisa dijangkau?
  • Seberapa cepat responnya?
  • Apakah koneksinya stabil?

üì¶ Bagaimana Ping Bekerja?

Ping menggunakan protokol yang bernama ICMP (Internet Control Message Protocol). Cara kerjanya:

  1. Komputer kamu mengirimkan paket yang disebut ICMP Echo Request ke target (misalnya IP 192.168.1.254).
  1. Target akan membalas dengan ICMP Echo Reply jika masih hidup dan bisa dijangkau.
  1. Ping mencatat waktu dari pengiriman hingga balasan diterima (round-trip time).

üñ•Ô∏è Contoh Perintah Ping

ping 192.168.1.254

Berarti kamu mengirim sinyal ke perangkat dengan IP 192.168.1.254, dan menunggu balasan.

  • 6 paket dikirim, 6 diterima, tidak ada yang hilang (0% packet loss)

    • avg: rata-rata waktu ‚Üí 4.160 ms
    • max: waktu terlama ‚Üí 10.313 ms
    • mdev: variasi (standard deviation) ‚Üí 2.864 ms
